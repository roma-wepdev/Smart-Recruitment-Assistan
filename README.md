# Smart Recruitment Assistant Using Machine Learning

## 📌 Project Overview
This project is an AI-powered recruitment assistant that predicts whether a candidate is likely to change jobs based on their profile. The goal is to help HR departments make faster and smarter hiring decisions using Machine Learning.

---

## 📊 Dataset
- Total Records: 19,158
- Features: 13
- Target Variable:
  - 0 = Not Looking for a Job Change
  - 1 = Looking for a Job Change

---

## 🧹 Data Preprocessing
- Removed unnecessary columns
- Handled missing values
- Label Encoding
- Feature Engineering
- Train-Test Split

---

## 🤖 Machine Learning Models
- Logistic Regression
- Random Forest
- XGBoost

---

## 📈 Model Performance

| Model | Accuracy |
|--------|----------|
| Logistic Regression | 76.54% |
| Random Forest | 76.59% |
| XGBoost | **76.96%** |
